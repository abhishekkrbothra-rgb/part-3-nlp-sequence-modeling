# Customer Support Text Classification
This project uses an LSTM (Long Short-Term Memory) neural network to automatically classify customer support queries into categories.

## Approach
- **Data Cleaning:** Removed special characters and converted text to lowercase.
- **Preprocessing:** Used Tokenization and Padding to prepare text for the model.
- **Model:** Built a Sequence Model using an Embedding layer and an LSTM layer.
- **Evaluation:** Monitored accuracy over 5 training cycles (epochs).

## Results
The model evaluation graph and sample predictions can be found in the `/results` folder.
